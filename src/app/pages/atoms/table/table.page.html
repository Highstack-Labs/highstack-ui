Cómo funciona

La tabla es data-driven: le pasas [data] y [columns] y pinta todo sola. Para los casos que no son texto plano hay dos mecanismos:

1. Dato plano

field: 'nombre' → pinta el valor tal cual.

2. Dato anidado

field: 'direccion.ciudad' → resuelve el path con punto.

3. Render custom

<ng-template tableCell="field"> → tu componente por celda.

La idea clave: no escribes una fila ni una celda a mano. Declaras las columnas en un array y, solo donde necesitas algo especial (una fecha con ícono, un badge de estado, botones de acción), añades un template asociado a ese field. Todo lo demás se pinta automáticamente.
