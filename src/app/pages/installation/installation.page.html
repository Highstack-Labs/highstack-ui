Introducción

@highstacklabs/ui es una colección de componentes de UI premium construidos para Angular 21/22 y diseñados para funcionar de forma nativa con Tailwind CSS v4. Todos los componentes son standalone y aprovechan las últimas características como Angular Signals para el control de estados reactivos.
